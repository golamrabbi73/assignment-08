# 🧱 Tiles Gallery

A modern and responsive Tile Gallery web application built with Next.js App Router. Users can explore beautiful tile collections, search tiles, view detailed information, and manage their profiles securely.

> This project was built as part of Assignment Category A8-Apple using Next.js App Router and modern authentication system.

---

## 🌐 Live Website

https://assignment-08-alpha.vercel.app

---

## 📌 Project Purpose

The purpose of this project is to create a visually appealing and user-friendly platform where users can browse different types of tiles, explore detailed information, and manage their accounts securely using authentication and protected routes.

---

## ✨ Features

- 🔥 Modern Responsive UI Design
- 🧱 Dynamic Tile Gallery from JSON/API
- 🔍 Search Tiles by Title
- ⭐ Featured Tiles Section (Top 4 Tiles)
- 🔐 Authentication System (BetterAuth)
- 🟢 Google Social Login
- 👤 User Profile Page
- ✏️ Update User Information (Name & Image)
- 🚦 Private Route Protection
- 📱 Fully Responsive for Mobile, Tablet & Desktop
- 🎞️ SwiperJS Integration
- 📢 Dynamic Marquee Section
- ❌ Custom Not Found Page
- ⏳ Loading Spinner & Better UX Handling

---

## 🛠️ Technologies Used

- Next.js 15 (App Router)
